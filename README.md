# 🌱 잔디 심고 갈래?
혼자 코딩하지만 함께 몰입하고 있다는 감각을 전달하는 GitHub 기반 온라인 모각코 서비스

## 💡 이런 서비스입니다

`늦은 밤, 혼자 코딩할 때 슬랙에서 동료의 파란 불을 보며 힘을 얻은 적 있으신가요?`

**잔디 심고 갈래?** 는 바로 그 느낌을 온라인 공간으로 옮겨왔습니다. **캠·마이크 없이** 같은 공간에서 함께 코딩하는 경험을 제공합니다.



### 핵심 가치
- 🌍 **공동의 성장**: 개발자들의 GitHub 활동이 모여 맵을 변화(예: 황무지를 숲으로 변화)시킵니다
- ⏱️ **집중의 시각화**: 커밋, PR, 집중 시간이 실시간으로 반영됩니다
- 🐾 **펫과 함께**: 활동으로 얻은 포인트로 펫을 뽑고 성장시킵니다
- 🤝 **느슨한 연대**: 캠/마이크 없이, 같은 공간에서 함께 코딩하는 경험을 제공합니다

---



## 🎯 이런 분들을 위한 서비스입니다

**홀로 작업하며 동기부여**를 원하거나, **오프라인 모각코는 부담**스럽지만 **함께하는 느낌**을 원하는 개발자

---

## 🆚 기존 서비스와의 차별점

| 기존 서비스 | 한계점 | **잔디 심고 갈래?** |
| --- | --- | --- |
| 온라인 모각코 | 진입장벽 (규칙, 화면/캠 공유) | **규칙 없이 즉시 참여** ✅ |
| 메타버스(ZEP, 게더타운) | 개발자 특화 기능 부재 | **GitHub 연동 개발자 특화** ✅ |
| 디스코드/슬랙 | 공간감·시각적 성취 부족 | **유대감·시각적 성취** ✅ |
| 오프라인 모각코 | 지리적 한계,<br/>내향적 성향 부담 | **온라인으로 가볍게 접속** ✅ |

#### 펫 획득 & 성장

```mermaid
flowchart LR
    A[포인트 모으기] --> B{사용 방법}
    B -->|가챠| C[새 펫 획득]
    B -->|밥주기| D[펫 경험치 +]
    D --> E{필요 경험치 도달?}
    E -->|Yes| F[진화 버튼 활성화]
    F --> G[1단계 → 2단계 → 3단계]
    C --> D
```

- **가챠**: 포인트로 랜덤 펫 뽑기
- **밥주기**: 보유 펫에게 포인트를 투자하여 경험치 누적
- **진화**: 필요 경험치 도달 시 다음 단계로 진화 (최대 3단계)
- **대표 펫**: 선택한 펫이 캐릭터를 따라다니며 동행

## 사용자 시나리오

```mermaid
flowchart TB
    subgraph 입장
        A[GitHub 로그인] --> B[가상 공간 입장]
        B --> C[캐릭터 이동 & 자리 잡기]
    end

    subgraph 작업
        C --> D[태스크 생성]
        D --> E[집중 타이머 시작]
        E --> F[개인 작업]
    end

    subgraph 성장["개인 & 공동 성장"]
        F --> G{활동 발생}
        G -->|커밋/PR| H[GitHub 감지]
        G -->|집중 30분| I[집중 시간 적립]
        G -->|태스크 완료| J[태스크 완료]

        H --> K[포인트 획득 + 게이지 상승]
        I --> K
        J --> K

        K --> L{게이지 100%?}
        L -->|Yes| M[🌲 맵 변화!]
        L -->|No| F
        M --> F
    end

    subgraph 펫["펫 시스템"]
        K --> N[포인트 누적]
        N --> O{펫 활용}
        O -->|가챠| P[새 펫 획득]
        O -->|밥주기| Q[펫 경험치 상승]
        Q --> R{진화 가능?}
        R -->|Yes| S[🐾 펫 진화!]
    end
```

```mermaid
flowchart TB
    subgraph Client ["Client (Browser)"]
        direction TB
        NextJS["Next.js (SSG)"]
        Phaser["Phaser 3 Game Engine"]
        Zustand["Zustand Store"]
    end

    subgraph Server ["Backend (NestJS)"]
        direction TB
        Auth["AuthModule<br/>GitHub OAuth + JWT"]
        Player["PlayerModule<br/>이동, 접속 시간"]
        GitHub["GithubModule<br/>GraphQL 폴링"]
        Room["RoomModule<br/>방 배정"]
        Task["TaskModule<br/>태스크 관리"]
    end

    subgraph DB ["Database"]
        SQLite["SQLite + TypeORM"]
    end

    subgraph External ["External"]
        GitHubAPI["GitHub GraphQL API"]
    end

    Client <-->|Socket.io| Server
    Client <-->|REST API| Server
    Server <--> DB
    GitHub <-->|30초 폴링| GitHubAPI
```
