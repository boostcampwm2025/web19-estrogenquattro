# 프로젝트 소개

🌱 잔디 심고 갈래? 서비스는

혼자 코딩하지만 함께 몰입하고 있다는 감각을 전달하는
GitHub 기반 온라인 모각코 서비스입니다.

개발자의 GitHub 활동(커밋, PR, 집중 시간)을 기반으로
개인의 성취를 공동의 성장으로 시각화하여
지속 가능한 동기부여와 유대감을 만듭니다.
