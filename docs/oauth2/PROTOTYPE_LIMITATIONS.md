# 프로토타입에서 생략된 부분 (추후 개선 필요)

빠른 프로토타입 구현을 위해 아래 항목들은 생략되었습니다. 프로덕션 환경에서는 반드시 고려해야 합니다.

## 1. 데이터베이스

| 현재 | 개선 방향 |
|-----|----------|
| 인메모리 (Map) | PostgreSQL, MySQL 등 영속성 DB |

**영향:**

- 서버 재시작 시 모든 사용자 데이터 소실
- 서버 수평 확장 시 사용자 데이터 공유 불가

## 2. Refresh Token

| 현재 | 개선 방향 |
|-----|----------|
| Access Token만 사용 | Access Token + Refresh Token |

**영향:**

- Access Token 만료 시 재로그인 필요
- 토큰 만료 시간을 길게 설정하면 보안 위험 증가

## 3. 기타 고려사항

- [ ] 토큰 블랙리스트 (강제 로그아웃용)
- [ ] Rate Limiting (브루트포스 방지)
- [ ] 로그인 이력 관리
- [ ] 다중 기기 로그인 관리
